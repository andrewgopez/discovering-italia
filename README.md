# Discovering Italia
Discovering Italia is a cultural and educational website project exploring the regions of Italy through geography, history, landmarks, food, language, and regional identity.

During the Winter 2025 semester at John Abbott College (CEGEP), I took a course called *Special Topics in Italian Culture*. I really enjoyed both the course and the way it was structured, and many of my classmates felt the same way. The experience inspired me to create a website dedicated to Italian culture and regional identity.

## Features
- Regional navigation system
- Dedicated pages for Italian regions
- Historical and cultural information
- Landmark cards with images
- Responsive card layouts
- Embedded videos and media
- Clean museum-inspired design

## Technologies Used
- HTML5
- CSS
